# E-Commerce Sales Dashboard

## Project Overview
This project analyzes e-commerce sales data and presents key business insights using an interactive dashboard.

## Tools Used
- Python (Pandas)
- Excel
- Looker Studio

## Key Insights
- Total Sales: ₹10.6M+
- Total Profit: ₹1.84M+
- Total Quantity Sold: 17K+
- Sales analysis by category, region, and month
- Identification of top selling products

## Process
1. Data cleaning and preprocessing using Python (Pandas)
2. Data preparation in Excel
3. Dashboard creation using Looker Studio

## Dashboard
This dashboard helps understand:
- Monthly sales trends
- Regional sales performance
- Product category profitability
